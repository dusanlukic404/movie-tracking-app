
# Flutter Firebase Movie Tracker

This Flutter application utilizes Firebase for the backend to provide users with functionalities such as signing in, signing up, resetting passwords, and managing their watched movies. Users can perform CRUD operations on movies, including adding, updating, and deleting them. Additionally, the app supports sorting, searching, and filtering movies by genre.

# Features

- User authentication (sign in, sign up, password reset)
- Movie management (add, update, delete)
- Movie listing with sorting, searching, and filtering by genre

# Technologies Used

- Flutter
- Firebase Authentication
- Firebase Firestore

# Setup Instructions

- Clone the repository.
- Set up a Firebase project and add your Firebase configuration to the Flutter project.
- Enable Firebase Authentication and Firestore in your Firebase project.
- Run the Flutter application on your preferred device or emulator.

# Usage

- Sign in or sign up using your email and password.
- Add movies that you have watched.
- Update or delete movies as needed.
- Use sorting, searching, and filtering options to manage your movie list effectively.
